# الكاتب الإبداعي بالذكاء الاصطناعي

## وصف المشروع
مسار تدريبي تجريبي غير ربحي لتعلم كيفية استخدام الذكاء الاصطناعي في الكتابة الإبداعية وإنتاج المحتوى.

## الميزات
- واجهة مستخدم متجاوبة باللغة العربية (RTL)
- 4 أسابيع من المحتوى التدريبي التفاعلي
- مشروع نهائي شامل
- موارد تعليمية متنوعة
- تصميم حديث باستخدام Tailwind CSS

## التقنيات المستخدمة
- React.js
- Vite
- Tailwind CSS
- PM2 (لإدارة العمليات)

## كيفية التشغيل

### التطوير
```bash
# تثبيت الحزم
npm install

# تشغيل الخادم المحلي
npm run dev
```

### باستخدام PM2
```bash
# بدء التطبيق
pm2 start ecosystem.config.cjs

# عرض الحالة
pm2 status

# عرض السجلات
pm2 logs creative-ai-course --nostream

# إيقاف التطبيق
pm2 stop creative-ai-course
```

## بنية المشروع
```
creative-ai-course/

```

## المحتوى التعليمي

### الأسبوع الأول: إتقان أدوات الكتابة بالذكاء الاصطناعي
- مقدمة إلى أدوات الذكاء الاصطناعي للنصوص
- مقارنة بين ChatGPT، Claude، Gemini
- كتابة أوامر (Prompts) فعّالة
- تحويل الأفكار إلى محتوى

### الأسبوع الثاني: أنواع المحتوى الإبداعي
- أنواع المحتوى المختلفة
- الفرق بين المحتوى التعليمي والترفيهي والتسويقي
- التركيب البصري والنصي

### الأسبوع الثالث: الأتمتة وإنتاج المحتوى
- مقدمة إلى الأتمتة
- ربط الأدوات في سير عمل متكامل
- أهمية المراجعة البشرية

### الأسبوع الرابع: بناء المشروع الشخصي
- تحويل المهارات إلى مشروع عملي
- بناء ملف تعريفي
- مشاركة المحتوى بمسؤولية
